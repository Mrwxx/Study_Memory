老规矩–妹妹镇楼：

一. merge算法

       将两个容器中的元素合并存储到另一个容器中。

       原始的两个容器必须是有序的，且顺序相同。

@param beg1 容器1开始迭代器
@param end1 容器1结束迭代器
@param beg2 容器2开始迭代器
@param end2 容器2结束迭代器
@param dest  目标容器开始迭代器

二．sort算法

       容器中的元素排序

@param beg 容器1开始迭代器
@param end 容器1结束迭代器
@param _callback 回调函数或者谓词(返回bool类型的函数对象)

三．random_shuffle算法

       对容器中的元素进行随机排序，即打乱元素的排序

@param beg 容器开始迭代器
@param end 容器结束迭代器

四．reverse算法

       将容器中的元素顺序反转

@param beg 容器开始迭代器
@param end 容器结束迭代器
