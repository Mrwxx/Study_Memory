## 一.	merge算法
&nbsp;  &nbsp;  &nbsp;  &nbsp;**将两个容器中的元素合并存储到另一个容器中。**

&nbsp;  &nbsp;  &nbsp;  &nbsp;原始的两个容器必须是**有序**的，且顺序相同。
<br>

```cpp
@param beg1 容器1开始迭代器
@param end1 容器1结束迭代器
@param beg2 容器2开始迭代器
@param end2 容器2结束迭代器
@param dest  目标容器开始迭代器
```
<br>
