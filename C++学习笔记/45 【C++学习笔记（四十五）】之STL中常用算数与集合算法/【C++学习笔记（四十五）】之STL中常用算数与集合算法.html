老规矩–妹妹镇楼：

一． accumulate算法

        计算容器元素累计总和

@param beg 容器开始迭代器
@param end 容器结束迭代器
@param value起始值

二． fill算法

        向容器中填充相同的元素 

@param beg 容器开始迭代器
@param end 容器结束迭代器
@param value t填充元素

三． set_intersection算法

       求两个容器元素的交集，容器元素必须有序。

@param beg1 容器1开始迭代器
@param end1 容器1结束迭代器
@param beg2 容器2开始迭代器
@param end2 容器2结束迭代器
@param dest  目标容器开始迭代器
@return 目标容器的最后一个元素的迭代器地址

四． set_union算法

       求两个容器元素的并集，容器元素必须有序

@param beg1 容器1开始迭代器
@param end1 容器1结束迭代器
@param beg2 容器2开始迭代器
@param end2 容器2结束迭代器
@param dest  目标容器开始迭代器
@return 目标容器的最后一个元素的迭代器地址

五． set_difference算法

       求两个容器元素的差集，容器元素必须有序

@param beg1 容器1开始迭代器
@param end1 容器1结束迭代器
@param beg2 容器2开始迭代器
@param end2 容器2结束迭代器
@param dest  目标容器开始迭代器
@return 目标容器的最后一个元素的迭代器地址
