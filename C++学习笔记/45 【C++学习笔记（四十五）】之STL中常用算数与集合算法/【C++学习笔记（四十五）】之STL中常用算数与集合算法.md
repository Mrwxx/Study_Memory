	@param beg 容器开始迭代器

## 二．	fill算法
 &nbsp;  &nbsp;  &nbsp;  &nbsp; 向容器中**填充相同的元素**	
<br>

	@param beg 容器开始迭代器
	@param end 容器结束迭代器
	@param value t填充元素
<br>

## 三．	set_intersection算法 
 &nbsp;  &nbsp;  &nbsp;  &nbsp;求两个容器元素的**交集**，容器元素必须有序。
<br>

	@param beg1 容器1开始迭代器
	@param end1 容器1结束迭代器
	@param beg2 容器2开始迭代器
	@param end2 容器2结束迭代器
	@param dest  目标容器开始迭代器
	@return 目标容器的最后一个元素的迭代器地址
<br>

## 四．	set_union算法 
 &nbsp;  &nbsp;  &nbsp;  &nbsp;求两个容器元素的**并集**，容器元素必须有序
<br>

	@param beg1 容器1开始迭代器
	@param end1 容器1结束迭代器
	@param beg2 容器2开始迭代器
	@param end2 容器2结束迭代器
	@param dest  目标容器开始迭代器
	@return 目标容器的最后一个元素的迭代器地址
<br>

	@param dest  目标容器开始迭代器
