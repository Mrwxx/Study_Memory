老规矩–妹妹镇楼：

一． Vim中的查找操作

（一）/ string

       从光标处开始查找string字符串，用n向下遍历，用N向上遍历

（二）? String

       从光标处开始查找string字符串，用n向上遍历，用N向下遍历，与/相反。

（三）#

       当光标在某个单词处时，我们直接用#键就可以查找该单词，遍历依然用n/N


二． Vim中的格式调整操作

（一） >>

       当前光标所在行右移一个tab

（二） n >>

       移动光标所在行开始的n行右移一个tab

（三） <<

       当前光标所在行左移一个tab

（四）n >>

       移动光标所在行开始的n行左移一个tab

（五） gg=G

       将整个文件一次性调整格式


三． Vim中查看man帮助文档


       在Vim中，光标位于想要查询的函数处，按Shift + 小写k查看该函数文档；按住n + 大写K转到第n页文档。

四． 命令模式转编辑模式

（一） 小写i

       在光标之前插入字符

（二） 大写I

       在光标所在的行首插入字符

（三） 小写a

       在光标之后插入字符

（四） 大写A

       在光标所在的行尾插入字符

（五） 小写o

       在光标所在的行的下一行插入字符

（六） 大写O

       在光标所在行的上一行插入字符

（七） 小写s

       删除光标所在的字符，变为插入模式

（八） 大写S

       删除光标所在的字符，变为插入模式

（九） ESC

       退出插入模式到命令模式
