老规矩–妹妹镇楼：

一． 最大公约数

（一）概述

       最大公约数即数a和数b之间的所有公约数中最大的那个公约数。求解最大公约数要用欧几里得算法（辗转相除法）：

       设a, b 均为正整数，则gcd(a, b) = gcd(b, a % b)

       gcd(a, b)为a和b的最大公约数

（二）思路

       欧几里得算法的思路就是不断地减小数据的规模，不断地通过比较a和b的大小，交换a和b，使得数据总体不断减小。

       当a < b时，则交换了a和b；

       当a > b时，使得数据减小到b以下，规模减小

当规模减小到什么程度时，得出结果?

       0和任意整数a的最大公约数都是a，这个就是最后的递归边界。

（三）代码实现

1. 递归式

gcd(a, b) = gcd(b, a%b)

2. 递归边界

gcd(a, 0) = a

代码如下：

int gcd(int a, int b) {
	if (b == 0)
		return a;
	else
		return gcd(b, a % b);
}
