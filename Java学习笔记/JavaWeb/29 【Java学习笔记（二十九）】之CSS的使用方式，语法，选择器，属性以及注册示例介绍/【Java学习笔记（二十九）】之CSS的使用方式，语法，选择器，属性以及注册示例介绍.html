一． CSS介绍

（一） 概述

       Cascading Style Sheets 层叠样式表，将多个样式作用在同一个html的元素上，同时生效。

（二） 优势

       将内容与样式的控制进行分离，降低耦合度，提高开发效率。

二． CSS的三种使用方式

（一）内联样式

       在内容的标签上使用style属性指定css代码。这种方式css代码只能作用于该代码段。
如：

<div style=”color:red;”>hello world</div>

（二）内部样式

       在head标签中定义style标签，style标签的标签体内容就是css代码。这种方式css样式只能作用于该HTML文件。

如：

<style>
		<div>{
			color:red;
		}
</style>

<div>hello world</.div>

（三）外部样式

       单独定义css资源文件，在html的head标签内定义link标签，引入外部的资源文件。这种方式css样式能够作用于所有的文件。Link标签中的href表示css文件的URL。

a.css文件：

div{
	color:green;
}

html文件：

<link rel=”stylesheet” href=”./a.css”>
<div> hello world</ div>

也可以用以下方式引入css文件：

<style>
	@import “./a.css”;
</style>

三． CSS的语法

（一） 格式

选择器{
	属性名1：属性值1；
	属性名2：属性值2；
}

（二）选择器

       选择器：筛选具有相似特征的元素。

1. 基础选择器

（1） Id选择器

       用”#”号选择具体的id属性值的元素，一个html页面中id值唯一

#id属性值{}

（2） 元素选择器

       选择具有相同标签名称的元素

标签名称{}

（3） 类选择器

       用”.”选择具有相同的class属性值的元素

.class属性值{}

（4） 选择器优先级

       id选择器>类选择器>元素选择器

2. 扩展选择器

（1） 并集选择器
选择器1，选择器2{}

       中间没有空格，两个选择器都选中。


（2） 子选择器

       筛选选择器1元素下的所有选择器2元素，中间有空格。如 a b{}，则选择a的所有b后代元素。

选择器1 选择器2{}

（3） 父选择器

       筛选选择器1的子选择器2，只选择一代，中间用>号。如 a > b{}，则只选择a的后一代b元素。

选择器1 > 选择器2{}

（4） 属性选择器

       选择元素名称，属性名=属性值的元素。

元素名称[属性名=“属性值”]{}

（5） 伪类选择器

       每个元素都有不同的状态分为

 初始化状态link
 被访问过的状态visited
 正在访问状态active
 鼠标悬浮状态hover

       可以通过伪类选择器设置元素在不同的状态的样式。

元素：状态{}
a:link{}

（二） 属性

1. 文本

font-size:字体大小
color: 文本颜色
text-align: 对齐方式
line-height:行高

2. 背景

background

3. 边框

border:设置边框属性

4. 尺寸

width:宽度
height:高度

5. 盒子模型

       用于控制布局。

margin:外边距，即该盒子与上一级的盒子的距离
padding:内边距，即该盒子与同级的盒子之间的距离
float：浮动，可让多个盒子浮动到一行
